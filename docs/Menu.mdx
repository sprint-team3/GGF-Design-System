## Title

- Menu

## Description

- A component for displaying a menu with clickable items.

## Anatomy

- **`menuList`**: An array of objects representing menu items, each containing:
  - **`menu`**: Text representing the menu item.
  - **`path`**: URL path associated with the menu item.
- **`activatedMenu`**: Index of the currently activated menu item.
- **`handleActivatedMenu`**: Function triggered when a menu item is activated, taking the index of the activated menu item as an argument.

## Usage

```jsx
import { Menu } from 'ggf-ui';
```

```jsx
<Menu
  menuList={[
    { menu: 'Home', path: '/' },
    { menu: 'About', path: '/about' },
    // Add more menu items as needed
  ]}
  activatedMenu={0}
  handleActivatedMenu={handleActivatedMenu}
/>
```
